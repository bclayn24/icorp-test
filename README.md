# ICORP Test

### Цель проекта
Скрипт для взаимодействия с тестовым API, демонстрирующий навыки HTTP-коммуникации и обработки данных.

### Установка и запуск скрипта

1.  **Клонирование репозитория:**
    ```bash
    git clone https://github.com/bclayn24/icorp-test &&
    cd icorp-test
    ```

2.  **Установка зависимостей:**
    ```bash
    npm install
    ```

3.  **Запуск ngrok:**
    Запустите туннель на порту 3000, чтобы внешний API мог обратиться к вашему локальному серверу.
    ```bash
    ngrok http 3000
    ```

4.  **Конфигурация WEBHOOK_URL:**
    В файле проекта замените заглушку в переменной `WEBHOOK_URL` на **ваш публичный HTTPS-домен**, предоставленный ngrok.

    > **Пример:**
    > Замените адрес в коде: `const WEBHOOK_URL = "https://<ваш-ngrok-домен>.ngrok-free.dev/get/second/part";`

5.  **Запуск скрипта:**
    ```bash 
    npm run dev
    ```
